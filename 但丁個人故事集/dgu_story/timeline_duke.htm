本銀河曆的用法：每五千年進一單位，如：100.4999的下一年是101.0000。

故事的編號乃參考本銀河曆的年份而來，但亦可進行微調。

　

　

本銀河曆１００.４６０５年：

紅獄神女的羽生莎織號召成立「精算者小隊」。

　　　　│

本銀河曆１００.４６３０年：（冥神家的女兒們）

冥神夜誕生於「金牛座子象限第966星系」第六養殖行星，身高超過三百公里。誕生兩個月後，第六行星滅亡，70億個基因改造種喪生。

被洗去出生記憶的冥神夜，發現自己身處在某個無名黑洞的引力圈中。

　　　　│

本銀河曆１００.４６５１年：（夢幻豔陽天）

銀河各地不斷出現假冒里斯本水手的詐騙案件，無數受欺騙而上當的人類女性自中央星團被流放。

　　　　│

本銀河曆１００.４６７２年：（炸彈嘉年華）

精算者小隊的羽生莎織前往波勒帝星系調查離奇的爆炸事件。

　　　　│

本銀河曆１００.４６９８年：（禍在今朝）

以朵蘿希亞為首的一群殘酷系神女，從東方象限的隔離設施逃獄成功。包括海泓星系在內的十多個人類社會受到她們的襲擊。

　　　　│

本銀河曆１００.４７０３年：（世界撲滅之三人）

因為契約失效，小可憐駕駛「紅淚」離開沮洳星系，但此時的法雷爾已不以任何身份存在，小可憐獲得自由。

沮洳星系毀滅，但因位置偏僻，這件悲劇從未被本地銀河的人類社會知悉。

　　　　│

本銀河曆１００.４７１１年：

中央銀河星團象限的「第一天京星系」與「布蘭森共和國」之間爆發星際戰爭，有人類主權機關為盟友的布蘭森共和國很快就佔了上風。

　　　　│

本銀河曆１００.４７１２年：（亞述皇國異聞錄）

紅獄神女之一的拉菲亞在「里奧661-9星系」上甦醒，不久後被賽露休絲與同樣身為紅獄神女的小可憐找到，三者決定協力建國。

亞述皇國在北方銀河象限宣告獨立，並將根據地里奧661-9星系改名為「亞述星系」。

第一天京星系輸掉了與布蘭森共和國的星際戰爭，被迫向神女一族求援，因此召開了星系爭奪戰。

亞述皇國在最後關頭加入第一天京星系的爭奪戰，名義上雖未勝利，但成功獲取了各方勢力的承認，亞述皇國就此走向穩定發展的道路。

娜塔莎被第三代的波爾維克大公爵選定為接班人選之一。

　　　　│

本銀河曆１００.４７１８年：（穿梭艇）

某個星系的大氣層發生了一段小小故事。

　　　　│

本銀河曆１００.４７２５年：（童年好時光）

北方象限的札絲緹星系遭到神女襲擊，整個星球超過半數的成年人遇害。

　　　　│

本銀河曆１００.４７５０年：

接受朵蘿商會的邀請，冥神夜來到唐布羅斯星系中的工業行星「凱維蘭帝IV」，隨後不明原因地開始以人類的身份過著教師生活。

　　　　│

本銀河曆１００.４７６０年：

卡斯特．路德在中央銀河星團的巴洛亞星系出生 。

娜塔莎宣誓就任第四代的波爾維克大公爵、卸下原本玫瑰修士會會長的職務。

精算者小隊結束階段性任務，就此停止活動、解散成員，發起者羽生莎織移居至東方象限。

　　　　│

本銀河曆１００.４７６６年：（冥神家的女兒們）

冥神靜誕生於「金牛座966星系」第七養殖行星，有超過1200名同伴，是冥神家數量最多的一代。

　　　　│

本銀河曆１００.４７８０年：

冥神夜以人類女子「夜」的身份前往星團大學進行交流訪問，並初次與卡斯特．路德相遇。

　　　　│

本銀河曆１００.４７８１年：（至黑之夜）

在本地銀河四處尋訪有關故鄉線索的希薇爾，與來自韋瓦第星系的術譜調律師傑克．賈維斯相遇，兩人因故同行。

金穗銀河事件發生。

　　　　│

本銀河曆１００.４７８ ６年：

卡斯特路德與希薇爾初次相遇。

在得知自己還有兩位素昧平生、同母異父的兄弟之後，卡斯特路德解封了身上藏有的術譜之力，並自認為法雷爾的唯一合法繼承者。

卡斯特路德與希薇爾暫時別離。

　　　　│

本銀河曆１００.４７８７年：（鍋蓋裡的戰爭、天國覺醒）

在星團大學研究人類生物學的神女奧菲莉雅，在科隆星系的戰亂中得知研究生卡斯特路德 的真正身分，從此決心跟隨。

卡斯特路德與奧菲莉雅離開星團大學。

　　　　│

本銀河曆１００.４７８８年：（鍋蓋裡的戰爭）

卡斯特路德與奧菲莉雅來到唐布羅斯星系，欲吸收冥神夜為事業夥伴，未果。

唐布羅斯星系中的工業行星「凱維蘭帝IV」遭受嚴重破壞，負責管理的朵蘿商會亦傷亡慘重，被懷疑是犯案者的冥神夜就此遭到通緝。

　　　　│

本銀河曆１００.４７８９年：

波爾戴綠旗軍的武星翠加入卡斯特路德的計畫，同時希薇爾回歸。

　　　　│

本銀河曆１００.４７９０年：

姬海娜商社的前身企業「姬海娜物流公司」創立，初始成員有董事長「卡斯特．路德」、執行長「希薇爾」、財務長「奧菲莉雅」與業務部長「武星翠」。

甫成立的姬海娜物流公司捲入北方象限的大型企業戰爭，三個有人星系在僅僅四十天的戰爭裡毀滅。

卡斯特路德獲得來自天原商社的正式授勳，成為路德公爵，但並沒有受封采邑。

　　　　│

本銀河曆１００.４７９５年：（雪佩雅絲）

奧士里茨星系會戰爆發，別稱「三王會戰」、「繼承人之戰」，最終獲勝者是卡斯特路德。

　　　　│

本銀河曆１００.４８００年：

姬海娜物流公司將營業登記地址正式遷移至姬海娜星系，並以該星系所遺留的遠古遺跡「戴森球體」為基礎繼續建設要塞，得到飛越性成長。

年底，姬海娜物流公司改組，宣布成立上市企業「姬海娜商社」。

　　　　│

本銀河曆１００.４８０７年：（兀爾德之星）

法雷爾以路德公爵的身份，介入北方銀河象限「第九出雲星系」的文明發展，改變了當地的政治版圖。

　　　　│

本銀河曆１００.４８１６年：（冥神家的女兒們）

冥神靜與其餘1200位同伴離開遭到滅亡的「金牛座966星系」的第七行星， 並失去對故鄉的記憶。

　　　　│

本銀河曆１００.４８１７年：（兀爾德之星）

紅獄神女北岡千鶴在自己故鄉的第九出雲星系，使用了術譜「三女神的午茶座」，將故鄉以百年前的樣貌加以替換。

　　　　│

本銀河曆１００.４８ ２５年：

長期與姬海娜商社敵對的漢諾威經合同盟陷入名存實亡的狀態；布羅迪亞聯邦瓦解，其背後也有姬海娜商社活動的影子。

　　　　│

本銀河曆１００.４８５０年：

卡斯特社長出席了有超過九千個人類社會協議組成的「獨立星系聯邦」建國大會，儘管該國普遍被認為是姬海娜商社的附庸組織。

　　　　│

本銀河曆１００.４８５１年：

人類主權機關正式終止與姬海娜商社的外交關係，並沒收國境內所有的商社資產。

姬海娜商社的決策核心進行改制，引入皇室制度，希薇爾等神女成為皇女，但卡斯特仍保持社長頭銜。

　　　　│

本銀河曆１００.４８６７年：

肇因於對環形世界色雷斯的主權聲索，姬海娜商社與天原商社在教廷協調下達成協議，劃定非戰爭的緩衝區。

　　　　│

本銀河曆１００.４８９１年：（天國覺醒）

20歲的柴可夫通過試煉，成為姬海娜商社的社員，跟隨二皇女奧菲莉雅。

　　　　│

本銀河曆１００.４８９５年：（最終神子）

在北方象限塞奧芝亞星系，術譜系統「命運石王座」挑選出神聖統合聯邦的最後繼承者 仲宇桓。

　　　　│

本銀河曆１００.４８９６年：（天國覺醒）

擔任姬海娜二皇女奧菲莉雅秘書的柴可夫，與人類女子結婚。

　　　　│

本銀河曆１００.４８９７年：（最終神子）

仲宇桓從塞奧芝亞星系脫出，為了貫徹正義理想而開始獵殺殘酷系神女。

　　　　│

本銀河曆１００.４８９８年：（狼入虎口）

教廷的一位玫瑰修士成功潛入神秘的橡樹島俱樂部進行蒐證，但隨即失去聯絡，事後證實其為背叛教廷、自稱「悖律復仇者」的文素。

　　　　│

本銀河曆１００.４９００年：（天國覺醒、橡樹島的獵物）

路德公爵在橡樹島與背叛教廷的修士「悖律復仇者」文素初次見面，並陷入後者佈置的陷阱 而死亡，致使遠古惡魔的力量暴走，而文素亦被殺死37次。

橡樹島事件發生，274名神女與神使 在俱樂部內遭到屠戮，塔娜博士駕船逃往遠西銀河。當天，芙蕾雅從天原商社出走。

姬海娜商社的社長陷入植物人狀態 ，引發北方象限的激烈動盪，在商社內部也爆發分裂與接班爭議。

薩多船團與姬海娜商社三皇女武星翠率領的艦隊爆發區域衝突，前者被 突然出現的法雷爾所殲滅。

亞種神女莎蘭希．賽齊爾所領導的「快樂島」商會承包了橡樹島俱樂部的善後工作，與教廷大公爵娜塔莎建立起親密關係。

人類主權機關的雅各賓級二號艦「聖茹斯特號」，在多米尼克星系擊殺了14名追蹤著法雷爾行跡的前天原神使，造成銀河情勢的緊繃。

千年艦隊與姬海娜的四皇女絲汀娜，出兵追擊出現在遠西銀河邊緣的法雷爾，前者被三級魔艦「市場花園」給殲滅。

29歲的柴可夫離開姬海娜商社，前往人類主權機關的領域創業，改名柴．柴可夫。

　　　　│

本銀河曆１００.４９０８年：（橡樹島的獵物）

塔娜博士犧牲座艦、驚險逃過路德公爵的追擊，隨後抵達遠西銀河象限與中央銀河星團交界處的「天威星系」。

　　　　│

本銀河曆１００.４９１０年：

鐵士朗在北方銀河與黑物質之海的邊緣被涅妮特救起，結束宇宙漂流者的生活。

北方銀河象限的小國「亞述皇國」，遭到以姬海娜商社為首的眾勢力圍攻而滅亡，二代皇帝賽露休絲向姬海娜投降。

　　　　│

本銀河曆１００.４９２８年：（橡樹島的獵物）

塔娜博士在遠西銀河象限的「天威星系」被法雷爾找到並殺害。其所搭乘的「比例尺」宇宙船讓當地青年當上威政王，對地表的戰爭行為進行壓制。

　　　　│

本銀河曆１００.４９４０年：

路德公爵的延壽體耗盡所有能量，在魔艦「花園」上 永久地死去，法雷爾這一世代的生命到此結束。

　　　　│
