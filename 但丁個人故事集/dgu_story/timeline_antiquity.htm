本銀河曆的用法：每五千年進一單位，如：100.4999的下一年是101.0000。

故事的編號乃參考本銀河曆的年份而來，但亦可進行微調。

　

　

宇宙生成∼距今３０億年前前後

三次元向量的宇宙開始有生命萌芽，數十億年內由原始的單細胞藻類，演化成堪稱達到完美境界的超級生命。

　　　　│

３０億年前：

宇宙迎來第一次「大滅絕」時代，無數生命與種族快速消失，彷彿自始即不曾存在，原因至今不明。

　　　　│

１０億年前：

通稱「母海」的高位元能量開始流入三次元宇宙，但尚未發生任何影響。此時的宇宙被認為沒有生命存在。

　　　　│

１億年前：

宇宙再次誕生大量生命，包括史上最早的智人，即現今遍布全宇宙之人類的祖先。

　　　　│

５０００萬年前

遠古人類經歷無數次繁榮與滅亡，在銀河之間進行漫長的跨世代航行，一度逼近滅絕。

母海能量在三次元宇宙的濃度終於高到能以生命之姿顯現，但其真貌至今不明，是否具有智慧也未知。

　　　　│

５萬年前

人類終於成為能夠稱霸宇宙的優勢物種，建立橫跨一百億光年的人類帝國。

　　　　│

　　　　│

本銀河曆０９８.０００１年　前後：

人類的擴張腳步終於踏進本地銀河，載運生命種子的自動播種船不斷尋找適合的行星登陸，將星球進行宜居化改造。

　　　　│

本銀河曆０９８.３００１年　前後：

真正搭載有人類船員的移民船開始進入本地銀河，在傳說中頗負盛名的船隻有「拜占庭號」、「高天原號」及「盤古號」等。

　　　　│

本銀河曆０９８.４００１年　前後：

未知的災難摧毀了黑海航道，使本地銀河的人類文明與深太空的母國斷絕聯繫。無數殖民地失去超光速科技，部份星球甚至退化回蠻荒時代。

　　　　│

本銀河曆０９９.０００１年　前後：

被稱為「神女」的巨大女性生命體開始在本地銀河活躍，她們擁有個體進行超光速跳躍旅行的能力。

北方象限的入雲星系首次出現神女與人類比鄰而居的情況，國家由皇帝進行統治。

　　　　│

本銀河曆０９９.１００１年　前後：

自稱機械種族的全自動智能船艦首次出現在入雲星系。

在機械種族的協助下，入雲星的科學家建立了名為基底棧橋的蟲洞理論，重新恢復了超光速科技。

　　　　│

本銀河曆０９９.１５０１年　前後：

人口爆炸的入雲行星開始借助超光速科技，向數千個星系進行移民，並發現很多失落殖民地。

因統治力量在初期未能跟上擴張腳步，遙遠的殖民地開始自行宣布獨立，先後成立十二個郡國。

　　　　│

本銀河曆０９９.２００１年　前後：

高天原帝國在北方象限成立，聯合版圖橫跨六千光年的十二個郡國，成為本地銀河進入歷史時代後的第一個星際國家。

神女一族逐漸喪失自體進行超光速跳躍的能力，轉而尋求機械種族的協助，命定座艦制度自此時開始，被稱為「古老的盟約」。

（天原商社的社史從這一年代開始記載，這也是本地銀河現存最古老的可靠史料）

　　　　│

本銀河曆０９９.３００１年　前後：

更多的超光速理論被實驗成功，宇宙船艦進入私有化時代，前所未見的拓荒熱潮持續超過一千年。

由於星系之間的治安問題難以解決，部份人類選擇進行有去無回的長途遷徙，進入塞維尼亞銀河系。

　　　　│

本銀河曆０９９.４００１年　前後：

經過長久的競爭與兼併，銀河的東方象限出現比高天原更大規模的星際國家：拜占庭神女帝國。

由於神女的降生現象越發普遍，東方象限的人類逐漸在政治上喪失主導權。

　　　　│

本銀河曆１００.０００１年　前後：

為了使各個星系自行建立的時間觀念有所一致，拜占庭開始推廣本銀河曆法與公制度量衡，逐漸成為全銀河系的標準體制。

殖民風潮開始深入此前缺乏開發的南部銀河象限，眾多零碎的小國被建立。

　　　　│

本銀河曆１００.１０００年：

愛德蒙出生。

　　　　│

本銀河曆１００.１０２５年：

第一次神團戰爭在本地銀河的遠西象限爆發，相關史料留存甚微。

數年後，神團被逐出本地銀河， 遠西象限從此成為星路斷絕的密境，漫長而辛苦的航道復原工作在數百年後才開始。

　　　　│

本銀河曆１００.１３６１年：

５月４日

一小股在第一次神團戰爭裡潰逃的神團分艦隊，花了三百年的時間才離開遠西象限，並進入北方象限的斯托尼亞星區，大量血族神女的領地被捲入戰火。

　　　　│

５月２５日

支撐已到極限的神團分艦隊，將一件重要物品封印於斯托尼亞星區的某處之後，全體自爆而滅亡。

　　　　│

５月２７日

與神團的抵抗戰爭宣告結束，焦土化的斯托尼亞星區被高天原合併而成為一省，血族神女的領袖阿爾緹妮絲被冊封為樞黎親王。

　　　　│

本銀河曆１００.１４２１年：（魔獸的圖章）

樞黎星系爆發神團再次入侵的危機。

　　　　│

本銀河曆１００.２００１年　前後：

波爾戴軍事聯盟在北方象限宣告成立，其前身是溫柔系神女海賊的互助會組織。

漢莎艦隊商盟在東方象限一間小型貿易站的音樂廳裡宣告成立，創始人之一是來自北方象限的愛莉亞．馮．德爾敦克．諾福克 ，其並自封為公爵。

　　　　│

本銀河曆１００.２１２１年：

高天原帝國的神女皇帝遭到反對派暗殺，首都毀於一旦，六成以上的高天原文明產物在此一事變中消滅。

事變前後一共有六個郡國及四個行省曾短暫宣告獨立。

　　　　│

本銀河曆１００.２２９６年

９月１７日

採礦船「武仙座力士號」於南部象限的礦業星系亞德利亞邊境，與探路的第三神團先遣艦隊發生接觸。

　　　　│

１０月１日

亞德利亞星系遭神團分艦隊毀滅，20萬居民喪生，這是第二次神團戰爭最初的傷亡。

　　　　│

１０月１０日

羅格茲尼星系與舒姆達星系遭神團分艦隊毀滅，共4000萬居民喪生，調查人員發現入侵的勢力不只一個。

有關舒姆達星系毀滅的報告當日即被提交給拜占庭神女帝國，但由於帝國行政體系的效率低落，這份報告遭到了忽視。

　　　　│

１０月２２日

由於有越來越多的殖民地失聯，拜占庭神女帝國審判庭的檢察官佐久夜接下了調查的任務。

　　　　│

本銀河曆１００.２２９７年

１月１日

在絕大多數南部象限的人類社會都未得知有敵對勢力入侵的情況下，慶祝元旦的慶典在各地舉行。

超過十個的小型殖民地在歡慶新年的活動中遭神團毀滅。

　　　　│

１月５日

佐久夜檢察官判斷事態嚴重，迅速向上級提交了報告書，要求更大權限的調度與指揮權力。

　　　　│

１月６日

總部設在蒙迪德摩星系的星域安全理事會終於收到來自皮托亞星系航運總督對「武仙座力士號」在亞德利亞目擊不明艦艇的報告。

佐久夜檢察官的報告書仍然未受到拜占庭高層的重視，派遣大規模艦隊的要求遭到了回絕。

　　　　│

１月２２日

第一代波爾維克大公爵雪莉奴．波爾維克決定支持佐久夜，於本日派遣自己的私人艦隊與後者共同前往蒙迪德摩星系。

　　　　│

１月２５日

佐久夜與大公國艦隊進入蒙迪德摩星系，但該星系已被第九神團滅亡，50億居民死亡或不知去向。

佐久夜將身份不明的敵人取了「亞德利亞」的代號，通用至今。

　　　　│

２月１４日

人類政府態度軟化，決定與神女合作。超過八千名神女聚集在可能受襲的三個星域，以人類為主的奧林匹克協約諸國亦積極出兵。

佐久夜調查後，發現包括圖斯庫爾、第三天龍、菲思雅各等十六個在星圖上被標記的有人星系，已經被敵人滅亡。

　　　　│

２月２７日

大公國艦隊捕捉到一艘第三神團落單的受損戰艦，這是本地銀河首次得知入侵勢力的面貌。

佐久夜決定對南部象限的人類行星進行大撤退，將靠近危險地區的居民強制遷徙，因缺乏溝通導致此舉在日後招致了怨恨。

　　　　│

３月３日∼３月４日

伊迪恩星域會戰爆發，奧林匹克協約諸國的人類艦隊遭第三神團十六萬艘戰艦圍剿殲滅，本戰役證明人類完全無法對抗神團。

拜占庭帝國的琦絲嘉子爵與部下為了殿後而犧牲，成為亞德利亞神團戰爭最初的神女死者。

　　　　│

３月上旬至中旬

大撤退的範圍被要求擴張，超過一千個有人星系被強制疏散，銀河各地的術譜調律師開始加入戰鬥。

　　　　│

３月２０日

佐久夜將琦絲嘉子爵從伊迪恩戰場上送回的訊息解讀完畢，證實入侵勢力為自稱「神團」、正體不明的深太空種族。

公孫祿教授提供的資料顯示，這已經是神團第二次大規模入侵本地銀河，上一次發生在遠西象限。

　　　　│

３月２４日

人類方面的議和代表搭乘五艘無武裝的禮賓艦，抵達有亞德利亞前哨艦隊出沒的綠沙洲星系，企圖單方面與神團講和。

　　　　│

３月２９日∼４月１日

和談失敗，神團聯軍對有重兵駐守的博學者星系與斯魯比特星系發動進攻。

指揮斯魯比特人類艦隊的荷比斯．盧少將不敵第九神團，下令拋棄當地居民逃出星系。

斯魯比特僅剩拜占庭帝國的林菱伯爵率領少數神女進行頑抗，最終全滅。

　　　　│

４月３日

博學者星域會戰結束，在初代波爾維克大公爵雪莉奴與人類術譜調律師的奮勇作戰下，大敗第三神團。

北方象限的高天原帝國派遣7000名神女南下參戰。

　　　　│

４月１０日

佐久夜與雪莉奴率領4000名神女組成的大軍，對斯魯比特展開反攻，迫使第九神團的蟲群分艦隊逃向伊迪恩星系。

羨慕大公爵的豐碩戰果，其它的神女軍團蠢蠢欲動，渴望擊敗強大的敵人來彰顯實力。

　　　　│

４月１５日

波爾戴軍事聯盟違抗聯軍指揮部的命令，私自進攻伊迪恩星系。第二次伊迪恩星域會戰或稱伊迪恩反擊戰爆發。

最新的第十六神團忽然出現，她們擁有複製對手術譜能力的本領，使波爾戴軍的整體傷亡率超過七成。

受命殿後的白旗軍團全員未歸，伊迪恩反擊戰以神女的大敗收場。

此時代的法雷爾開始有所動作。

　　　　│

４月２０日（史稱「慟哭之日」）

第十六神團「墮落天女」越過戰區前線，同時進攻洛德紅峰、思善、第六威靈頓等位在後方的十八個星系，500億居民喪生或失蹤。

　　　　│

４月２２日

因第十六神團的外貌及行為都過於類似神女，聯軍內部爆發矛盾。同日還有四個星系受到「墮落天女」的攻擊。

絕望的人類政府開始組織大規模移民船隊，準備永久放棄南部象限。

　　　　│

４月２４日

高天原帝國軍採取新的戰術行動，分批駐守在500個人口超過一億人的大型星系；來自北方的神女已增員到超過一萬名。

　　　　│

５月

亞德利亞三支神團偃旗息鼓，整個月沒有發動攻擊。

高天原帝國軍派駐在500個星系上的神女，因長期相處而開始與當地居民發生摩擦，與神團完全無關的人類傷亡報告開始傳出。

佐久夜檢察官返回東方象限，集中全力研究所有與神團相關的資料，企圖找出致勝方法。

　　　　│

６月（本地銀河史上最悲慘的一個月，又稱「斷腸之月」）

６月１日

一批前往金牛座７４號星系的聯軍偵查艦隊未照原訂計畫返航。

　　　　│

６月３日

「黑蝙蝠級」偵查艦在廣達兩萬光年的三十六個不同星區，同時發現神團的活動跡象，聯軍的大前線顯然已被突破。

　　　　│

６月４日

位在包索羅星系的高天原帝國戰場司令部遭三支神團圍攻，高天原皇帝五天銖真奈美率眾反擊，成功擊退神團，但包索羅行星的人類居民慘遭全滅。

皇帝下令追擊神團。五小時後，所有高天原神女都接到聖旨，受命前往神無陵星系集合，但大公爵雪莉奴質疑這個命令的真實性，並阻止了部份的北方神女前去會合。

　　　　│

６月６日

「神無陵慘劇」發生，高天原總帥與7400名神女下落不明，座艦均被毀壞。高天原帝國為捍衛尊嚴，宣佈皇帝以下全員戰死，南部象限的戰線瀕臨崩潰。

　　　　│

６月７日

神團發動「三頭犬大進擊」，第九神團進攻普洛士星系，第三神團進攻智德星系，在三日之內聯軍超過70個重要據點失去聯絡，八日之內多達180個有人星系遭到攻陷。

　　　　│

６月１５日

以南部象限為傳統根據地的亞種神女「翼族」決意逃亡，她們帶上根據地的人類進入超遠距離蟲洞，從此被本地銀河的神女視為背叛者。

　　　　│

６月２０日

淪陷於神團魔爪的有人星系，總數突破300個，死亡或失蹤的人類數量超過六百億。

　　　　│

６月２３日

五角菱星域會戰爆發，第十六神團初次嚐到敗績，與第九神團一起受到重大打擊。

對行徑怪異的私人遊船「基督山號」的目擊報告開始傳出。

　　　　│

６月２７日∼６月２９日

奧林匹克協約諸國防衛戰爆發，二十個有人行星遭到焦土攻擊，兩百億人民化為宇宙塵埃，協約諸國滅亡。

人類方面大受打擊，多數政府將貴重的戰艦從前線調回，加入快馬加鞭的撤退作業，超過五百個殖民地被放棄而成為鬼城。

更多有關「基督山號」及「伯爵」的目擊報告陸續從淪陷區傳出，神團的軍勢似乎正在被削弱。

　　　　│

６月３０日

帝國審判庭檢察官沙耶香所率領的民間避難艦隊，在參宿七附近失去聯絡，認定已陣亡。（悲嘆的封印）

　　　　│

７月１日

一支由第三與第九神團組成、目標為東方象限的大艦隊，規模幾乎是神團本次戰爭投入總量的一半，在航經太陽系的時候神秘消失。

自願趕來本地銀河支援戰事的冥神愛遭到波及而被摧毀其存在、靈魂被打碎到廣佈整個南部象限。

　　　　│

７月４日

「墮落天女」正式以擁有自我意識的個體為使節，向拜占庭遞交停戰書，並返還被她們俘虜的一百五十億人類。

愛德蒙當眾揭穿了「伯爵」的真面目。

第一代波爾維克大公爵雪莉奴．波爾維克戰死。「基督山 」爆炸沉沒，愛德蒙與伯爵不知去向。

高天原帝國滅亡。

漢莎艦隊在疆域擴大的天原商社領地大為活躍。

東方象限的拜占庭神女帝國逐漸衰落，各地陷入航運的大停滯時代，星際交流一蹶不振。

共榮法案帶來的弊端開始顯現，精於政治算計的人類開始在帝國中樞逐漸取代神女的位置，神女帝國變得有名無實。
