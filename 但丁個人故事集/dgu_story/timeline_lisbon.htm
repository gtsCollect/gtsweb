本銀河曆的用法：每五千年進一單位，如：100.4999的下一年是101.0000。

故事的編號乃參考本銀河曆的年份而來，但亦可進行微調。

　

　

本銀河曆１００.４０２４年：（里斯本夜曲：墮落的美姬）

波賽特．萊昂．法雷爾在里斯布利奧行星出生，卻近乎是植物人狀態。

有「墮落美姬」稱號的芙蕾雅自教廷叛逃出走，投入天原商社旗下。

　　　　│

本銀河曆１００.４０３４年：（里斯本夜曲：亡靈的血契）

波賽特．萊昂．法雷爾在十歲生日時，完成了繼承法雷爾之名的全部所需條件，肉身因此得到甦醒。

芙蕾雅暫時離開本地銀河。

　　　　│

本銀河曆１００.４０３６年：

北境的札達爾王國遭到希貝尼克共和國的侵略而滅亡。

其末代王儲葛蕾西雅成為紅獄神女，隨即對希貝尼克展開了報復。

　　　　│

本銀河曆１００.４０４４年：

波賽特．萊昂．法雷爾宣誓加入 神怒會，並將每月的薪水全額匯給父母。

法雷爾對葛蕾西雅的欠款數字來到五百萬。

　　　　│

本銀河曆１００.４０４８年：（蟲與獸）

北境加利西亞星區的培格斯行星滅亡，神聖統合聯邦隨後在此殖民，抹去了培格斯行星先民的所有紀錄。

史提夫．佩斯卓與雪莉．佩斯卓兄妹自培格斯逃出，成為神怒會所屬的法雷爾部下。

法雷爾對葛蕾西雅的欠款數字突破五千萬。

　　　　│

本銀河曆１００.４０５２年：（泥流人間）

北境巴達霍斯星區的羅索行星滅亡，並於同年被神聖統合聯邦接管，改名柏布爾星系。

法雷爾對葛蕾西雅的欠款數字兩度突破一億。

葛蕾西雅拒絕再借給法雷爾任何一毛錢。

　　　　│

本銀河曆１００.４０７１年：

新銳戰船「和諧級」完成試航， 神怒會開始推動代號為「金蘋果」的計畫。

神聖統合聯邦進行組織改造，此後凡屬於宗教類事務，一律歸「崇拜部」管轄。

　　　　│

本銀河曆１００.４０７４年：（惡德的門徒）

波賽特．萊昂．法雷爾的父母先後過世。

神聖統合聯邦爆發史上最大的金融風暴，當局派出艦隊前往鎮壓。

北境西威斯星區的天都行星滅亡，當地遭到神聖統合聯邦接管，改名凱爾伍尼星系。

羽生莎織成為紅獄神女。

　　　　│

本銀河曆１００.４０７５年：（六日之劫）

北境卡斯提爾星區的麥迪遜行星滅亡，並於同年被神聖統合聯邦接管，改名蘭佩杜沙星系。

　　　　│

本銀河曆１００.４０９０年　前後：

神聖統合聯邦內部出現各種傳言，表示軍方正在對神女使用的母海能量加以的研究。

先進的加萊塞星戰船完成第四次升級，聯邦持有的總數超過一千九百艘，聯邦海軍的規模達到史上最大。

　　　　│

本銀河曆１００.４０９７年：

波賽特．萊昂．法雷爾隻身一人逃出北方象限，流亡到東方的教廷領域。

波賽特．萊昂．法雷爾開始自稱「里斯本水手」。

　　　　│

本銀河曆１００.４０９９年：

天原商社與教廷結盟，發動「冬雨行動」。

神女聯軍開始進攻神聖統合聯邦。

　　　　│

本銀河曆１００.４１００年

１０月１０日：（獵殺封鎖線）

拉古萊特會戰爆發，神女聯軍獲得決定性勝利。

　　　　│

１０月１７日：

勒班陀會戰爆發 ，神女聯軍失去六支艦隊，這場戰役可能是本地銀河歷史上神女死傷人數最多的單一戰鬥。

　　　　│

１１月３０日：

姬諾瓦星系背叛神聖統合聯邦，陷入暴動的行星地表發生大屠殺，聯邦公民遭屠戮殆盡。

　　　　│

本銀河曆１００.４１０１年：

２月１６日：

神女聯軍攻入威尼西亞星系， 主恆星爆炸，神聖統合聯邦亡國。

　　　　│

該年稍晚

天原商社佔據了一部分神聖統合聯邦的國土，成立達沃與青林行省。教廷雖提出抗議，卻不被理會。

里斯本水手毀滅了神怒會總部，將教廷亟欲取得的諸多資料銷毀，之後離開了荒無人煙的北境。

　　　　│

本銀河曆１００.４１５０年　前後：

擁有超過兩千年歷史的漢莎艦隊商盟正式宣告解散。最後一筆由貿易委員會簽訂的合約，內容是有關殯葬用品的交易。

　　　　│

本銀河曆１００.４２１９年：

妲綺絲等數十名曾經服役於第六艦隊的神女在北方象限發現了塞奧芝亞星系，並定居於此。

　　　　│

本銀河曆１００.４２６５年：

八個行星共用軌道的特殊星系在南部銀河被發現，命名為「金牛座子象限第966星系」，其中有五個有人行星。

　　　　│

本銀河曆１００.４２９９年：

南部銀河象限爆發跨星系級的重大災難「仙后星雲海嘯事件」，韋瓦第星系的人類研究術譜活動因此而開始受到壓制。

銀河社會對人類術譜調律師的歧視更加嚴重。

　　　　│

本銀河曆１００.４３０１年　前後：

亞種神女賽齊爾族來到本地銀河，建立蟲洞並展開通商事業，讓賽齊爾銀河逐漸成為本地最知名的衛星銀河系。

　　　　│

本銀河曆１００.４５１９年：（惡夢甦醒）

北方銀河的「富山９８星系」遭到里斯本水手襲擊，儘管有六名教廷的玫瑰修士協助阻止，但仍然走向滅亡。

里斯本水手將旗下紅獄名單之一的「拉菲亞．狄．亞絲路妮亞」 丟給星系居民文素，後二人開始逃亡。

文素決意向法雷爾復仇。

　　　　│

本銀河曆１００.４５２０年：

拉菲亞被丟棄於北方銀河的「里奧661-9星系」(即今日的亞述星系)。文素成為教廷玫瑰會的修士，開始學習術譜的操作與編寫。

　　　　│

本銀河曆１００.４５２３年：（世界撲滅之三人）

魔艦「紅淚」在進行跳躍航行時發生事故，意外陷進沮洳行星的地底。

里斯本水手因此與當地的三位住民簽下契約，並派遣紅獄名單之一的「小可憐」駐留於此。

　　　　│

本銀河曆１００.４６００年　前後：

里斯本水手忽然停止一切活動，被認為已經死亡。

一部分的紅獄神女組成「精算者小隊」，調查並收拾由里斯本水手造成的各種離奇事件。

　　　　│
